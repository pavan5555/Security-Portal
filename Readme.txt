Project: Security Portal
Language: Node js
Features: 
	Digital Signing
	Key Generation
	Hashing
	Steganography
	Watermarking

How to use:

	Digital Signing: Upload a Document(pdf) and the signed document with public and private key(passphrase : pavan) are generated.
	Key Generation:	Click Generate to Generate a Public and Private Key pair.
	Hashing: Enter Input text to find the MD5 hash.
	Steganography: Enter text and select image click hide to encode text into image and to download it, Upload the downloaded image and hit Retrive to get back the hidden text .
	Watermarking: Select image and hit submit, the image with watermark will get downloaded.
